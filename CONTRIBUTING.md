# Contributing to Opengaebi

Thank you for your interest in Opengaebi.

## Current Status

**This project is not accepting external code contributions at this time.**

We are in early development and focusing on establishing a stable foundation before opening up to community contributions. This policy will be revisited as the project matures.

## How You Can Help

While we are not accepting pull requests, the following are very welcome:

- **Bug reports** — Open an issue describing the problem, steps to reproduce, and your environment
- **Feature requests** — Open an issue describing the use case and why it would be valuable
- **Questions** — Open a discussion if you are unsure whether something is a bug or expected behavior

## Issue Guidelines

When opening an issue, please include:

- Opengaebi version (`./bridge --version` or Docker image tag)
- OS and architecture
- Go version (if building from source)
- Minimal reproduction steps
